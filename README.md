# Building Local LLM Desktop Applications with Tauri

Hey there!

As you may know, building with local models can be challenging but the ecosystem is getting better! I wrote this repo as a little Sunday afternoon hacking project to try and capture an approach for how application developers can actually go about building native LLM applications for Macs, Windows and Linux with hopefully a somewhat familiar JS/TS tooling system (i.e. Tauri).

llama.cpp is one of my most favourite open source projects out there and provides a great runtime for local models. Here is an article I wrote that walks through how to build local model desktop apps based on this repo: https://medium.com/@dillon.desilva/building-local-lm-desktop-applications-with-tauri-f54c628b13d9
